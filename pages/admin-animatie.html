⚠️ Storage Bucket Setup Vereist

De storage bucket "intro-animations" bestaat nog niet. Maak deze aan via een van de volgende methoden:

Optie 1: Via Supabase Dashboard

  1. Ga naar je Supabase project → Storage
  2. Klik op New bucket
  3. Naam: intro-animations
  4. Public bucket: Aan
  5. File size limit: 104857600 (100MB)
  6. Allowed MIME types: video/mp4
  7. Klik op Create bucket

Optie 2: Via SQL Editor

  1. Ga naar je Supabase project → SQL Editor
  2. Kopieer de inhoud van supabase/migrations/create_animations_storage_bucket.sql
  3. Plak en voer de SQL uit
